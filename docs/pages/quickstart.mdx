# Quickstart

**Prerequisites:**

* **Docker:** Ensure Docker (Docker Engine & Docker Compose) is installed and running on your system.

**Steps:**

1. Clone the DocsGPT repository:

   ```bash
   git clone https://github.com/Nilzkool/DocsGPT.git
   cd DocsGPT
   ```

2. Start DocsGPT using Docker Compose:

   ```bash
   docker compose -f deployment/docker-compose.yaml up -d
   ```

3. Access the DocsGPT web application:

   Open your browser and navigate to [http://localhost:5173/](http://localhost:5173/)

4. Stopping DocsGPT:

   ```bash
   docker compose -f deployment/docker-compose.yaml down
   ```

For development workflows or advanced deployment options, see our [Docker Deployment documentation](/Deploying/Docker-Deploying). For advanced configuration, refer to [DocsGPT Settings](/Deploying/DocsGPT-Settings).
